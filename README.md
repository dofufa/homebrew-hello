# homebrew-hello
hello world homebrew repo

# INSTALL

1. Install [Homebrew](https://brew.sh).

2. Add this repo as a brew tap:

```bash
brew tap dofufa/hello
```

3. Install this formula:
```bash
brew install dofufa/hello/hello
```

OR, the latest:
```bash
brew install --head dofufa/hello/hello
```
